📱 Instructions pour mobile

Pour utiliser le microphone sur votre téléphone :

  1. Appuyez sur "Autoriser" quand le navigateur le demande
  2. Si bloqué :
    • Android : Paramètres → Apps → Chrome → Autorisations → Microphone
    • iOS : Réglages → Safari → Microphone
  3. Redémarrez l'application après avoir modifié les paramètres
