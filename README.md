# Plaud - Notes Vocales

Une application web progressive (PWA) pour enregistrer et organiser vos notes vocales avec une gestion optimisée des permissions microphone sur mobile.

## 🚀 Fonctionnalités

- **Enregistrement audio** avec visualisation en temps réel
- **Gestion des dossiers** pour organiser vos enregistrements
- **Recherche et tri** avancés
- **Mode hors ligne** grâce à la PWA
- **Compatibilité multi-plateforme** (Android/iOS/PC)
- **Gestion intelligente des permissions** microphone

## 📱 Problèmes d'autorisation microphone ?

### Solutions pour Android

1. **Dans Chrome** :
   - Allez dans `chrome://settings/siteSettings/microphone`
   - Trouvez votre site et sélectionnez "Autoriser"
   - Redémarrez Chrome

2. **Dans les paramètres système** :
   - Paramètres → Apps → Chrome → Autorisations → Microphone
   - Activez l'autorisation
   - Redémarrez votre téléphone

### Solutions pour iOS

1. **Dans Safari** :
   - Réglages → Safari → Microphone
   - Activez l'autorisation pour votre site

2. **Dans les paramètres système** :
   - Réglages → Confidentialité → Microphone
   - Activez Safari
